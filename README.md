# One Billion Row Challenge - Módulo 9 de Engenharia da Computação

## Descrição do Desafio

A última atividade ponderada do módulo 9 de Engenharia da Computação é resolver o One Billion Row Challenge. Essa tarefa consiste em escrever um programa para recuperar valores de medição de temperatura de um arquivo de texto e calcular a temperatura mínima, média e máxima por estação meteorológica. No entanto, há um desafio adicional: o arquivo possui 1.000.000.000 de linhas. O arquivo de texto tem uma estrutura simples com um valor de medição por linha, seguido pela estação meteorológica e a temperatura, separados por ponto e vírgula. O programa deve imprimir os valores mínimos, médios e máximos por estação, ordenados alfabeticamente.

## Solução em Go
A solução para este desafio foi implementada utilizando a linguagem de programação Go. Inicialmente, foi testada uma solução simplista, que apenas parseava cada linha do arquivo, calculava as medidas estatísticas e organizava o output. No entanto, essa abordagem se provou extremamente ineficiente, passando de meia-hora para rodar. Pensando nisso, resolvemos paralelizar o processo em 10 goroutines, atingindo a execução em 2m58s. Pensando nisso, a solução escolhida, de melhor performance, realiza as seguintes etapas:

1. Abre o arquivo de 1 bilhão de linhas no arquivo measurements.txt
2. Cria um arquivo para salvar os outputs em output.txt
3. Separa esse arquivo em 10 partes, para serem processadas paralelamente
4. Em cada processamento paralelo, acessa linha por linha e calcula as estatísticas parciais, isto é, o mínimo, máximo e a soma, além da running count de linhas analisadas. Esses valores são calculados com base no nome da estação, salvando em um hashmap.
5. Após os processamentos paralelos terminarem, os valores calculados separadamente sobre cada estação são comparados e utilizados para calcular o mínimo, máximo e média totais.
6. Enfim, as estatísticas de cada estação são organizadas em ordem alfabética, juntamente do nome da estação. Isso tudo é salvo em output.txt
